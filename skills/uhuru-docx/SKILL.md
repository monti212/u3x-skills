# Uhuru DOCX

Uhuru DOCX is the Uhuru-branded document skill manifest for Word document generation and editing workflows.

This path is intended to become the stable skill surface for the Uhuru 3 model family.

## Compatible Models

- Uhuru 3
- U3.4
- U3.5

## Migration Note

The repository may still contain legacy `kimi-*` skill paths during migration. New integrations should prefer `uhuru-docx`.
